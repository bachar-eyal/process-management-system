# Enterprise Asset & Process Tag Management System

A robust, full-stack desktop and web application built with Python, JavaScript, HTML5, CSS3, and SQLite for managing hardware assets, repair processes, and operational tags in air-gapped / offline environments.

---

## 🌟 Key Features

* **Multi-Tenant Architecture:** Isolated databases supporting separate team operations and workflow tracking.
* **Role-Based Access Control (RBAC):** Granular permission management for administrators, team leaders, and operators.
* **Asset Tracking & Barcode Integration:** Instant retrieval of hardware specs and service history via barcode scanning logic.
* **Interactive Data Analytics:** Real-time manager dashboards and visual charts to monitor team performance and inventory turnarounds.
* **Embedded AI Assistant:** Offline-ready query helper providing instant support and status lookups.
* **Air-Gapped Ready:** 100% offline functionality designed for high-security environments.

---

## 🛠️ Tech Stack

* **Backend:** Python, SQLite
* **Frontend:** HTML5, CSS3, JavaScript
* **Database & Auth:** SQLite, Multi-tenant Data Separation, RBAC Systems
